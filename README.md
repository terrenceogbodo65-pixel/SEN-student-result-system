# Student Result Management System

## Project Description
This project is a simple Python-based Student Result Management System developed for SEN 201 assignment.

## SDLC Explanation

### 1. Planning
The goal of this project is to develop a system that manages students' academic results.

### 2. Requirements Analysis
- Add student name and score
- Display student results
- Calculate average score
- Determine pass or fail

### 3. System Design
The system uses a Python file named `student_result.py`.
Functions used:
- add_student()
- display_students()
- calculate_average()

### 4. Implementation
The system is implemented using Python programming language.

### 5. Testing
The system was tested with multiple student scores to ensure accuracy.

### 6. Deployment
The project is deployed on GitHub and can be run on any system with Python installed.

### 7. Maintenance
Future updates may include database storage and a graphical user interface.
